# Statuses module

## Description

Adds statuses rotation, allows to set own status and more.

## Cogs

### `premiumControl` (**main**)

**Options** (optional):

- `options`
  - **`whoCanGive`** ([`string[]`][string], **optional**): IDs of additional users who can issue premium subscriptions.
